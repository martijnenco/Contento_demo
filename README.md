This app was a code demo for a company on angel.co.

Contento app

To install the app:
-----------------
To install the backend
###### `cd backend/`
###### ` npm i`
To install the frontend
###### `cd frontend/`
###### `npm i`

To start the app:
-----------------
###### `cd backend/`
###### `npm start`
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

###### `cd frontend/`
###### `npm start`
